# 股票分析系统

## 简介
股票分析系统是一个用于分析股票数据的工具，它可以帮助用户获取股票数据，并进行各种分析。该系统支持从Excel文件中读取股票数据，并提供了多种分析功能，如计算股票的收益率、计算股票的波动率等。

## 功能
- 从Excel文件中读取股票数据

## 运行示例
```shell
cargo run --example read_excel ~/works/RustroverProjects/stock/examples/fixtures/excel/test.xlsx test
```
